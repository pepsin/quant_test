移动平均 MA
Simple Moving Average — 价格基准线

计算最近 N 个收盘价的算术均值，作为衡量"价格中枢"的基准。策略中用 20 日均线判断乖离率、企稳信号。

公式
MAn = mean( close[t-n+1 .. t] ) = 1/n × Σ close[i]
// 代码：np.mean(close_series[-g.ma_period:]) 默认 n=20

乖离率 Bias
价格偏离均线的百分比 — 震荡期触发条件 ①

乖离率衡量当前价格与均线的相对距离。偏离过大说明短期涨势过猛，策略切换至更保守的高斯滤波器（震荡期）。触发阈值：bias > 10%。

公式
Bias = ( P − MAn ) / MAn
// 触发条件：Bias > bias_threshold（默认 0.10 = 10%）

RSI — 相对强弱指数
Relative Strength Index — 震荡期触发条件 ②

RSI 衡量一段时间内涨跌力度对比，取值 0–100。策略条件：前日 RSI > 75（超买）且今日 RSI < 60（回落）时切入震荡期。

公式
ΔP[t] = P[t] − P[t-1]
G = mean( ΔP[ΔP>0], last n )    // 平均涨幅
L = mean( |ΔP[ΔP<0]|, last n )   // 平均跌幅
RS = G / L
RSI = 100 − 100 / (1 + RS)
// 触发：prev_RSI > 75 AND RSI < 60

4
拉普拉斯滤波器
Laplace Filter — 正常期使用，EMA 变体

本质上是指数加权移动平均（EMA）的一种参数化方式。参数 s 控制敏感度，α = 1 − e−s。"拉普拉斯"之名来自其权重函数 w(k) = α·(1-α)^k ∝ e-sk 与离散拉普拉斯（指数）分布同形。

公式
α = 1 − exp(−s)   // 默认 s=0.05 → α≈0.0488
L[0] = P[0]
L[t] = α × P[t] + (1 − α) × L[t−1]   // 递推 EMA
// 信号通过：P[-1] > L[-1] AND L[-1] - L[-2] > 0.001

高斯滤波器
Gaussian Filter — 震荡期使用，更平滑

以距当前点的"距离"为自变量的高斯函数作为权重，离最新点越远的价格权重越小，衰减比指数更快（尾部更短），过滤震荡噪音效果更强。

公式
// k = 距当前点的距离（k=1 为最新，k=n 为最早）
w(k) = exp( −k² / (2σ²) )   // 默认 σ=1.2
G = Σ w(k) × P[t−k+1] / Σ w(k)   // 归一化加权均值
// 信号通过：P[-1] > G[-1] AND G[-1] - G[-2] > 0.002

对数价格加权线性回归 WLS
Weighted Least Squares on log(price) — 动量斜率提取

对对数价格序列做加权最小二乘拟合直线，权重从旧到新线性增大（1→2），使近期数据更重要。斜率 slope 代表对数收益增长率，是年化收益率的基础。

公式
y[i] = ln( P[i] )    x[i] = i     w[i] = linspace(1, 2, n)
// 最小化：min Σ w[i]·(y[i] - (a·x[i] + b))²
Sw = Σw,   Swx = Σw·x,   Swy = Σw·y,   Swxy = Σw·x·y,   Swx2 = Σw·x²
slope = (Sw·Swxy − Swx·Swy) / (Sw·Swx2 − Swx²)

7
R² 决定系数
Coefficient of Determination — 趋势一致性评分

R² 衡量回归直线能解释多少数据的方差。R²=1 完美趋势，R²≈0 纯噪音。策略用 R² 给动量"打折"，震荡走势不会获得高分。

公式
SS_res = Σ w[i] × (y[i] − ŷ[i])²   // 残差加权平方和
SS_tot = Σ w[i] × (y[i] − ȳ)²     // 总加权方差
R² = 1 − SS_res / SS_tot

8
年化收益率
两种计算方法 — 回归法 vs 简单收益法

策略用两种年化方式：长期用回归斜率（更稳定），短期用简单收益率复利化（响应更快），两者都用于过滤和排名。

公式
方法一：回归斜率（长期动量，默认 25 日）
annualized = exp( slope × 250 ) − 1
// 250 = 年交易日数；slope 来自对数价格回归
方法二：简单收益复利化（短期动量，默认 10 日）
r = P[−1] / P[−N−1] − 1
short_ann = (1 + r)250/N − 1

综合得分 Score
ETF 排名核心公式

得分 = 年化收益率 × R²。高收益但趋势杂乱的 ETF 得分会被 R² 折扣；低收益但趋势稳定的同样不高。策略只买得分最高的那一只。

公式
Score = annualized_returns × R²
// 高年化 + 高趋势质量 → 高得分

最大回撤 Drawdown
从近期高点跌落幅度 — 盈利保护触发依据

盈利保护：若当前价格低于近 N 日最高价 × (1−阈值)，则卖出持仓。震荡期退出条件之一：从近 20 日低点上涨 ≥ 3%。

公式
max_high = max( high[last N] )
drawdown = ( max_high − P ) / max_high
// 盈利保护触发：P ≤ max_high × (1 − threshold) 默认 5%
rise_from_low = ( P − min_low ) / min_low
// 退出震荡期：rise_from_low ≥ 3%

溢价率 Premium Rate
场内价格 vs 基金净值 — ETF 套利风险过滤

LOF/QDII ETF 场内价格可能高于净值（溢价）。溢价过高意味着价格虚高，策略排除溢价率超过 20% 的标的。

公式
premium_rate = ( market_price − NAV ) / NAV
// 过滤条件：premium_rate > 0.20（20%）则排除该 ETF
// 折价时为负数，不过滤
