# ETF 轮动策略风险分析

> 基于代码逻辑的客观推演，按严重程度从高到低排列。

---

## 第一级：可能造成重大亏损的情况

### 1. 隔夜黑天鹅 + T+1 双杀

```
13:11 买入某 ETF（全仓）
↓
收盘后重大利空消息（政策突变、地缘事件）
↓
次日开盘直接低开 8%～10%
↓
T+1：当天买的今天卖不了
↓
11:00 盈利保护触发 → 卖出
↓
但此时已经亏损 8%+，且可能继续跌
```

这是最危险的场景。策略完全没有盘后风险的防御手段。

---

### 2. 连续跌停，出不来

代码里明确写了：

```python
elif diff < 0:  # 卖出方向
    if data[security].last_price <= data[security].low_limit:
        log.info(f"{security} {name} 跌停，跳过卖出")
        return False
```

```
ETF 跌停 → 策略跳过卖出
第二天继续跌停 → 继续跳过
第三天继续跌停 → 继续跳过
→ 全程被锁死，只能眼睁睁看着亏损扩大
```

A 股 ETF 理论上没有涨跌停限制，但 QDII 类（纳指、标普等）和部分跨市场产品在极端行情下确实出现过连续大跌。

---

### 3. 动量选出"最高位"标的，然后急速反转

动量策略的本质是**追涨**：score 最高的那只，一定是最近涨最猛的。

```
某商品 ETF 连涨 20 天 → score 极高 → 策略全仓买入
↓
第 21 天：商品期货夜盘暴跌（策略感知不到）
↓
次日开盘跳空低开
↓
"涨得最猛的跌得也最狠"
```

2022 年豆粕、2023 年原油都出现过这种情形。

---

## 第二级：慢性失血

### 4. 震荡期基准与持仓完全脱钩

```
沪深 300 平稳 → 策略判断"正常期" → 用宽松的拉普拉斯滤波器
↓
同期：策略持仓的是纳指 ETF
↓
美联储加息预期变化 → 纳指连跌 3 周
↓
沪深 300 毫无感知，震荡期机制一直不触发
↓
策略全程持有亏损仓位，不切换防御
```

---

### 5. 参数过拟合后的"样本外失效"

这是最隐蔽的坏情况，不会一次性爆仓，而是：

```
回测：年化 30%，夏普比 2.0，看起来完美
↓
实盘头 3 个月：小亏，以为正常波动
↓
半年后：累计亏 15%
↓
反查原因：市场进入了一种历史回测中没出现过的走势
（比如：高波动率 + 无趋势 + 频繁跳空）
↓
所有精心调校的参数全部失灵
```

---

## 第三级：小概率极端情况

### 6. 全池齐跌，防御 ETF 也出问题

```
系统性金融风险（类似 2020 年 3 月）
→ ETF 池所有标的暴跌
→ 策略切换到货币基金 511880
→ 货币基金本身极其安全，但：
   - 若平台出现技术故障
   - 或流动性极度紧张时赎回延迟
→ 资金短暂被锁
```

货币基金本身几乎不会亏损，但"切换进去"这个动作可能在极端行情下执行失败。

---

## 风险汇总

| 场景 | 单次最大可能亏损 | 发生概率 |
|------|----------------|---------|
| 隔夜黑天鹅 + T+1 | 10%～20% | 低，但不罕见 |
| 连续跌停锁仓 | 20%～30% | 极低 |
| 追高后急速反转 | 10%～15% | 中 |
| 震荡期基准错位 | 慢性 5%～10% | 中高 |
| 参数过拟合失效 | 长期跑输市场 | **高** |

**最可能遇到的最坏情况不是某一次暴亏，而是参数过拟合导致的长期慢性亏损**——因为它没有明显的"爆炸点"，很难及时察觉和止损。
